# Tilt

> A read-only market intelligence dashboard for Indian equities. Scans the Nifty 500 across 14 NSE sectoral indices using momentum + value-gap filters, surfaces high-conviction entry signals, and validates every recommendation through a backtested + unit-tested signal engine. Built end-to-end with Claude Code.

**Status:** in development. Full README — including live demo links, AI-tooling angle, backtest results, disclosed limitations, and quickstart — lands at step 13 of the build. See [`docs/SPEC.md`](docs/SPEC.md) for the design decisions and implementation order.
