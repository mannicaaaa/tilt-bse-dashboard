# Tilt frontend — v2

React + Tailwind dashboard generated by [Claude Design](https://claude.ai/design). No build step — opens directly as `index.html` (React UMD + Babel-in-browser + Tailwind CDN).

## What v2 changed vs v1

- **One hero screen** (`Today's Picks`) instead of five. Recommendations grouped into 3 lanes: **Strong** / **Momentum** / **Value**.
- **StockCard** primitive: each pick shows ticker, price, score breakdown, **pros & cons** (derived from real indicator values — no LLM).
- **MutualFundsSection**: shows which top Indian equity MFs hold the same picks Tilt surfaced. Smart-money cross-check.
- **Sector heatmap** compressed to a small expandable pill in the hero (was a full grid in v1).
- 3 screens total: Today, Portfolio, Stock detail. No Scan, no Backtest screen.

## Files

| File | Role |
|---|---|
| `index.html` | Entry. Loads React, Babel, Tailwind, scripts. Sets `TILT_API_BASE`. |
| `api.js` | Fetch wrapper for the FastAPI endpoints + adapters mapping live payloads → TiltV2 shape. |
| `data.jsx` | **Static config only.** LANES, MUTUAL_FUNDS, STRATEGY_BACKTEST. Market data is fetched live. |
| `icons.jsx` / `primitives.jsx` / `chrome.jsx` | Design system + StockCard / LaneSection / MutualFundsSection. |
| `screen-today.jsx` | The hero — 3 lanes + MFs + sector backdrop pill. |
| `screen-portfolio.jsx` | Holdings tagged Hold / Average / Sell. |
| `screen-stock.jsx` | Drill-down with chart + indicator panel + filter triggers. |
| `app.jsx` | Routing, live API wiring, refresh logic. |
| `tweaks-panel.jsx` | Live design-token controls (carried over from v1). |

## Live data flow

1. On mount, `app.jsx` calls `TiltAPI.fetchTiltV2()` which fans out to:
   - `GET /sectors/heatmap` → `SECTORS`
   - `GET /scan/recommendations` → `STOCKS` (with lane + pros/cons baked in)
   - `GET /portfolio` → `HOLDINGS`
2. Refresh button calls `POST /refresh` (forces yfinance fetch + recompute), then re-fans.
3. On API failure, a red error banner appears at the top of the page; screens render empty arrays — **no silent fallback to mock seed data**.

## Mutual funds disclosure

There is no public/free API for Indian mutual fund portfolio holdings. The 6-fund snapshot in `data.jsx` is **manually curated** from each AMC's monthly factsheet (publicly downloadable PDFs). Refresh cadence: monthly. Production upgrade path: Tickertape / ValueResearch paid API behind a `MFHoldingsProvider` interface — same pattern as our `MarketDataProvider` / `PortfolioProvider` abstractions.

## Local dev

```bash
# Backend
cd ..
.venv/bin/uvicorn tilt.api.app:app --reload --port 8000

# Edit index.html to point at local backend:
#   window.TILT_API_BASE = "http://localhost:8000";

# Frontend
cd frontend
python -m http.server 5173
open http://localhost:5173
```

## Why no build step

Claude Design ships React UMD + Babel-in-browser deliberately — `view-source` works, the interviewer can read everything. A Vite build is fine evolution if this grows past prototype.
